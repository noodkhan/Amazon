# Installation Steps



## Using npm

Run commands WINDOW 

1) ```npm install```

2) ```npm run dev```

## Or using yarn

Run commands for APPLE 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
